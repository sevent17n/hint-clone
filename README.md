## Getting Started

First, run the build command:

```bash
npm run build
# or
yarn dev
# or
pnpm dev
```

Secondly, run the production server:

```bash
npm run start
# or
yarn start
# or
pnpm start
```

In package.json file you can see full list of commands.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
